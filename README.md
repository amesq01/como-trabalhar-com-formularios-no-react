# Como trabalhar com formulários no React

### Como Rodar o projeto

```
git clone git@github.com:emersonbroga/como-trabalhar-com-formularios-no-react.git
cd como-trabalhar-com-fomularios-no-react
npm install
npm start
```

Esse código é apenas para fins educativos e foi formatado para exemplificar conceitos.
